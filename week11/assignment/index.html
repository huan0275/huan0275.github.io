## On the Grid Use what you have learned about CSS Grid to layout the fields in different patterns. Use the `:nth-child()` function to target specific blocks. **Note:** The `grid-gap` property has been set to `5px` for all fields. 1. Use the `display` property to create a grid of **1 column** and **3 rows** that matches the following layout.
![field 1](http://imdac.github.io/images/codepen/gridline-field-1.png) 2. Use the `display` and `grid-template-columns` properties to create a grid of **3 columns** and **1 row** that matches the following layout.
![field 2](http://imdac.github.io/images/codepen/gridline-field-2.png) 3. Use the `display`, `grid-template-columns` and `grid-template-rows` properties to create a grid of **4 columns** and **2 rows** that matches the following layout. The first row should have a height `50px` and the second row should have a height of `100px`.
![field 3](http://imdac.github.io/images/codepen/gridline-field-3.png) 4. Use the `display`, `grid-template-columns`, `justify-content` and `align-content` properties to create a grid of **3 columns** and **1 row** that matches the following layout. Each column should have a width of `50px`.
![field 4](http://imdac.github.io/images/codepen/gridline-field-4.png) 5. Use the `display`, `grid-template-columns`, `align-items`, and `align-self` properties to create a grid of **3 columns** and **1 row** that matches the following layout.
![field 5](http://imdac.github.io/images/codepen/gridline-field-5.png) 6. Use the `display`, `grid-template-columns`, `grid-template-rows`, and `grid-column` properties to create a grid with **3 columns** and **3 rows** that matches the following layout.
![field 6](http://imdac.github.io/images/codepen/gridline-field-6.png) 7. Use the `display`, `grid-template-columns`, `grid-template-rows`, `grid-column` and `grid-row` properties to create a grid with **4 columns** and **4 rows** that matches the following layout.
![field 7](http://imdac.github.io/images/codepen/gridline-field-7.png) 8.
